# 图片资源说明

本目录存放用于图像识别的按钮截图。

## 需要的图片

| 文件名 | 说明 | 截取位置 |
|-------|------|---------|
| `comment_button.png` | 写留言按钮 | 文章底部的"写留言"或评论图标 |
| `comment_input.png` | 留言输入框 | 点击写留言后出现的输入框 |
| `send_button.png` | 发送按钮 | 输入框旁边的"发送"按钮 |

## 截取方法

### 手动截取

1. 打开微信公众号文章
2. 滚动到底部找到"写留言"按钮
3. 使用截图工具截取按钮区域
4. 保存到 `assets/mac/` 目录（Mac）或 `assets/win/` 目录（Windows）

## 注意事项

- 截图要清晰，包含完整的按钮
- 不要截取太大的区域，只截取按钮本身
- 如果界面主题（深色/浅色）改变，需要重新截取
